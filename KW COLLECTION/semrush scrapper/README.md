# README

## Extension-এর Basic Requirements

- Google Chrome ব্রাউজার
- SEMrush অ্যাকাউন্ট

## Tool কী করে

- SEMrush-এ URL লোড করে
- কিওয়ার্ড, ভলিউম এবং ট্রাফিক স্ক্র্যাপ করে CSV ফাইল তৈরি করে

## Tool দিয়ে কী করতে হবে

- কিওয়ার্ড রিসার্চ
- ট্রাফিক অ্যানালাইসিস

## কিভাবে Run করতে হবে

1. Google Chrome-এ Extension ইনস্টল করুন。
2. SEMrush অ্যাকাউন্টে লগইন করুন。
3. Extension-এর popup থেকে "Start" বাটন চাপুন。
4. প্রক্রিয়া শেষ হলে "Download CSV" বাটন চাপুন。
5. CSV ফাইলটি আপনার পছন্দমত জায়গায় সংরক্ষণ করুন।

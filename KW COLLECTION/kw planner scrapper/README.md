Extension-এর Basic Requirements

- Google Chrome (latest) এবং সক্রিয় Google Ads অ্যাকাউন্ট লগইন থাকা।
- অনুমতি: `activeTab`, `storage`, `contextMenus`, `tabs` এবং `https://ads.google.com/*`।

Tool কী করে

- Google Ads Keyword Planner থেকে কীওয়ার্ড ও সার্চ ভলিউম স্ক্র্যাপ করে।
- ডেটা গ্রুপ করে CSV ফাইলে দ্রুত ডাউনলোড করায়।

Tool দিয়ে কী করতে হবে

- আপনার CSV থেকে কীওয়ার্ড লোড করে ব্যাচে সার্চ চালাতে হবে।
- “Keywords you provided” এবং “Keyword ideas” থেকে ফলাফল সংগ্রহ করতে হবে।

কিভাবে Run করতে হবে

1. Chrome-এ `chrome://extensions/` খুলে Developer mode অন করুন।
2. “Load unpacked” এ ক্লিক করে এই ফোল্ডারটি সিলেক্ট করুন।
3. Google Ads Keyword Planner পেজে যান (লগইন নিশ্চিত করুন)।
4. এক্সটেনশন আইকনে ক্লিক করুন; Start চাপুন, প্রয়োজনে Stop/Downlaod/Clear ব্যবহার করুন।
5. স্ক্র্যাপড রেজাল্ট CSV হিসেবে পপআপ থেকে ডাউনলোড হয়ে যাবে।
